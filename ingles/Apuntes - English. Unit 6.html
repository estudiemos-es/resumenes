Apuntes de inglés: Unidad 6. El tiempo

Introducción y contextualización

El tiempo es una dimensión fundamental a la que se hace referencia constantemente en la comunicación diaria. Comprender y utilizar correctamente las expresiones de tiempo es crucial en cualquier idioma, especialmente en inglés, para evitar malentendidos. Esta unidad se enfoca en desarrollar la capacidad de expresarse y entender a otros cuando se refieren a momentos específicos.

Glosario

Esta sección introduce y repasa expresiones temporales esenciales para referirse a eventos en el presente, pasado y futuro, además de las preposiciones de tiempo más comunes.

  • Expresiones de tiempo presente:
  • Now / right now: Ahora mismo, en este instante.
  • Today: Hoy.
  • At the moment: En este momento.
  • Every (day/week...): Cada (día/semana...).
  • On (Mondays...): Los (lunes...).
  • Expresiones de tiempo pasado:
  • Yesterday: Ayer.
  • Last (week/year/summer...): La (semana/año/verano...) pasada.
  • (Two years / months...) ago: Hace (dos años / meses...).
  • Expresiones de tiempo futuro:
  • Tomorrow: Mañana.
  • Next (year/week...): El (año/semana...) que viene.
  • Preposiciones de tiempo:
  • In: Se usa para periodos de tiempo amplios.
  • Estaciones del año: In summer.
  • Meses: In September.
  • Partes del día: In the morning (*Nota: at night es una excepción para la noche*).
  • On: Se usa para periodos más específicos.
  • Días de la semana: On Monday.
  • Fechas concretas: On September 7th.
  • At: Se usa para horas del reloj y periodos vacacionales de varios días.
  • Horas: At half past eight.
  • Vacaciones: At Christmas.

Listening

Speaking

El objetivo de esta sección es aprender a utilizar las expresiones de tiempo para resumir y destacar cuándo ocurrieron las acciones más relevantes. Esto facilita la comunicación de información básica a través de estructuras simples.

  • Ejemplo de narración de actividades:

El documento presenta un ejemplo donde se describe una semana pasada, utilizando expresiones como Last week, From Monday to Friday, in the morning, in the afternoon, On Monday and Wednesday, on Wednesday, this weekend, on Friday, the whole day, On Saturday. Se observa el uso de la preposición on antes de los días de la semana.

  • Ejercicio propuesto:

Practicar las expresiones de tiempo narrando lo que se hizo el verano o la semana pasada, siguiendo el formato del ejemplo y empleando diversas preposiciones y expresiones.

Reading

Writing

El enfoque de esta sección es la elaboración de resúmenes escritos sobre eventos pasados, aplicando las expresiones de tiempo aprendidas.

  • Ejemplo de resumen escrito:

Se proporciona un texto que describe la Navidad pasada, destacando expresiones temporales como Last Christmas, On Christmas Eve, The next morning, In the afternoon, Later that evening, The day after Christmas. Estas expresiones ayudan a situar las acciones en el tiempo de forma clara.

  • Ejercicio propuesto:

Describir lo que se hizo el verano pasado en un texto de entre 90 y 120 palabras, haciendo uso de las expresiones de tiempo.

Mediation

La mediación se aborda como la habilidad de transmitir información de una imagen o cartel a otra persona, ya sea de forma oral o escrita. La sección enseña a describir una imagen utilizando expresiones clave.

  • Ejemplo de descripción de una imagen:

Se ilustra cómo describir una imagen de turistas, subrayando frases útiles para la descripción, la localización, la inferencia y la expresión de incertidumbre:

  • In this picture I can see: Para introducir lo que se observa.
  • In the background: Para describir el fondo.
  • It seems to be: Para expresar una impresión sobre algo.
  • I think: Para expresar una opinión o inferencia.
  • They are maybe: Para indicar una posibilidad.
  • I believe: Para expresar una convicción.
  • because: Para justificar una afirmación.
  • They are wearing: Para describir la vestimenta.
  • so I think that it is not so hot: Para deducir una condición a partir de una observación.
  • it could be: Para expresar una posibilidad.
  • It is possible: Para indicar probabilidad.
  • although I am not sure: Para expresar incertidumbre.
  • Ejercicio propuesto:

Describir una imagen proporcionada, utilizando las expresiones señaladas en el ejemplo y añadiendo otras conocidas.

Grammar support: Past continuous

El pasado continuo es un tiempo verbal utilizado para referirse a acciones que estaban en progreso en un momento específico del pasado.

  • Formación: Requiere el verbo auxiliar `to be` en pasado (`was` o `were`) y el verbo principal terminado en `-ing`. (Se remite a la Unidad 2 para repasar el verbo `to be`).
  • Estructura:
  • Afirmativa: Sujeto + was/were + verbo-ing.
  • Ejemplo: I was having a shower. (El sujeto precede al verbo conjugado en el tiempo correspondiente, que consta del auxiliar `to be` en pasado y el verbo principal con `-ing`).
  • Negativa: Sujeto + wasn’t/weren’t + verbo-ing.
  • Ejemplo: I wasn’t having a shower. (La estructura se mantiene, pero el auxiliar `to be` se presenta en su forma negativa).
  • Interrogativa: Was/Were + sujeto + verbo-ing?
  • Ejemplo: Were you having a shower? (La estructura sigue el patrón clásico de pregunta: auxiliar + sujeto + verbo principal con `-ing`).
